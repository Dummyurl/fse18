Description

Use Moto 360 ambient light sensor for automatic adjustment feature support.
Sets the minimum auto brightness to avoid dark screens in low-light.

Smart Brightness automatically switch from auto brightness to manual using a custom configuration. In low-light the Smart Brightness change the brightness from auto to manual mode and set the screen brightness to the minimum brightness making it easy to read the clock.

Smart Brightness has been tested with the minimum brightness at 10% and auto brightness on. The results are a better experience in low-light.
