Description

Droidddle is a Dribbble.com client. designers share what they are doing.

YOU MUST HAVE DRIBBBLE ACCOUNT TO GET THE SHOT.
if you don't have droidddle account, you can register on https://dribbble.com, it free. And login from Droidddle app.
If you have "401 Unauthorized" issue, you can uninstall and reinstall again to fix this issue.
If still get 401 error, please go to https://dribbble.com/account/applications , and Revoke the droidddle app, Then login again from the droidddle app.

★★★★★Features:★★★★★
* Get notified if have new following shots
* Change the "Following check frequency" to "Never check" will turn off the notifications.
* Watch Face for Android Wear
* Support Muzei Live Wallpaper
* Search shots on Dribbble
* Create shot on Dribbble
* Browse, Comment, Like shots on Dribbble
* Zoom in/out image
* View animated GIFs
* Follow Players on Dribbble
* Create bucket on Dribbble
* Share shot on Dribbble
* Black theme to save power on AMOLED screens
* A lot of other functions wait for you exploring

HOW TO USE DROIDDDLE: http://thezinx.com/android/droidddle-dribbble-app-material-design/

★★★★★More to come:★★★★★
* Offline browse
* Tablet, TV

If you like it , please rate ★★★★★,
If do not like, please rate ★★★★★, and give me a feedback.
Thank you!!!!!

☆☆ About the LIKE action ☆☆

Because the Dribbble api rate limiting is 60 requests per minute. And every shot/comment need a request to check you like it or not.When you viewing/scrolling shots,doing this will go over the rate limit quickly, cause can not loading more shots.\nSo we don\'t check the shot/comment like status, if you want like a shot just click the like button; then click again you can unlike this shot.

WHAT IS DRIBBBLE?

What are you working on? Dribbble is a community of designers answering that question each day. Web designers, graphic designers, illustrators, icon artists, typographers, logo designers, and other creative types share small screenshots (shots) that show their work, process, and current projects.
