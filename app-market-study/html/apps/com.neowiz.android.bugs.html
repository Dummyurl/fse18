슈퍼사운드, 벅스!
국내최대 음원을 고음질로 경험하세요!

- 국내최대 고음질 FLAC 음원 서비스
- 무선환경에서도 고음질 음원을 무손실로 전송하는 DLNA, 크롬캐스트 지원
- 디지털 노이즈를 제거로 선명하고 깨꿋한 음질을 감상할 수 있는 음질향상솔루션 래드손(RADSONE) 국내유일 적용

[더 편리해진 기능]
▶ 테마/장르 메뉴분리 : 국내에서 가장 세분화된 장르구분과 테마로 개인마다 다양한 취향의 음악을 쉽게 탐색할 수 있습니다
▶ 내 음악 메뉴신설 : '보관함'과 ' 내 앨범'을 ' 내 음악'으로 통합하여 저장/좋아한 음악과 내 앨범을 한 곳에서 감상할 수 있습니다
▶ 뮤직포스트 : 기존 '스페셜'이 '뮤직포스트'로 변신! 악필남, 스낵뮤직등 음악과 관련된 볼거리/들을거리가 더 풍부해졌습니다


[아름답고 멋진 기능]
▶ 안드로이드 최적화 및 특화기능 :
- 국내 음악서비스 최초 Android Wear 지원! 스마트워치에서 편리하게 즐기세요!
- 끊김 없는 재생(Gapless Playback) 구현, 공백없이 부드럽게 다음 곡으로 이어지는 재생!
- 좌,우 스와이프를 통한 빠른 화면전환 등 안드로이드폰 최적화
▶ 간편가입 : 이메일, 페이스북, 페이코(Payco)중 편한 방법으로 로그인하세요
▶ 뮤직4U : 빅데이터 기반의 지능적인 개인화 음악추천! 지금까지 무려 2억 9천만곡이 재생되었어요!
▶ 최대음원 : 가요, 팝, OST, 힙합, 락, 인디, 동요, 클래식 등 국내최대 음원을 차트,최신앨범, 추천음악등으로 즐기세요
▶ 라디오 : 때로는 검색이 귀찮을때가 있죠. 라디오 메뉴에서 테마/분위기대로 듣고 취향대로 선곡 기준을 만드세요
▶ 뮤직PD : 여러분이 선곡한 앨범을 다른 유저에게 공개할 수 있는 서비스에요. 일 평균 20만 건이상 이용되고 있답니다
▶ 위젯 : 바탕화면에 벅스앱을 추가해 보세요. 앱을 열지 않고도 폰화면에서 바로 차트를 확인하고 재생할 수 있습니다
▶ 재생모드 선택 : 전곡 바로 재생? 선택 후 재생? 벅스앱에서는 개인의 취향대로 선택할 수 있습니다
▶ 데이터 요금절약 : 데이터 요금이 걱정되세요? 앱에서 와이파이 연결 시 저장, 스트리밍 시 캐싱 옵션을 이용해보세요
▶ 다양한 특화설정 : 재생모드, 재생목록, 음량자동조절, 음질/음향 선택, 데이터요금절약 옵션, 재생종료타이머, 오디오 포커스,
플레이어 배경 색상 변경, 잠금화면 테마

--------------------------------------------------------------------------

[주요기능 이용안내]

▶ 2가지 재생 모드, 입맛에 맞게 선택하세요!
-[설정 > 재생] 메뉴에서 선택할 수 있어요.
-전곡 바로 재생 모드 : 기존 방식 그대로, 곡을 누르는 즉시 재생!
-선택한 곡 재생 모드 : 곡 선택 후 하단의 [듣기] 버튼을 누르면 재생! 선택한 곡만 재생목록에 담깁니다.

▶ 음악 재생 중 끊김 현상이 자주 발생한다면?
-하나, 네트워크가 정상적으로 연결돼 있는지 확인해 보세요.
-둘, RADSONE 설정을 OFF로 바꿔 보세요.
-셋, 스트리밍 음질 설정을 MP3로 바꿔 보세요.

▶ 로그인을 했는데도 계속해서 로그인하라는 문구가 나온다면?
-앱을 재설치한 후 이용해 보세요. (단, 저장한 음악을 내 앨범에 담아 안전하게 백업한 후 앱을 재설치해 주세요.)


[앱 접근 권한 안내]
안드로이드 OS 6.0(마쉬멜로우) 이상이 탑재된 기기에서는 벅스 앱을 처음 실행할 때 필수적 접근 권한에 동의해야 합니다.
선택적 접근 권한은 동의하지 않아도 벅스 앱을 이용할 수 있습니다.

▶필수적 접근 권한
- 전화 : 전화가 왔을 때 음악을 멈추고 통화 종료 후 다시 재생하는 데 필요한 권한
- 저장 : 외장 메모리에 음원을 저장/ 다운로드하고 로컬 음원을 조회하는 데 필요한 권한
- 주소록 : 기기 정보 확인과 새 소식 알림을 위해 필요한 권한 (주소록은 조회하지 않으니 안심하세요!)

▶선택적 접근 권한
- 마이크 : 음악 검색을 위해 필요한 권한

※ 안드로이드 OS 6.0 미만 버전에 대한 안내
안드로이드 OS 6.0 미만이 탑재된 기기를 사용 중이면 앱 접근 권한에 대한 동의 여부를 개별적으로 선택할 수 없습니다.
소프트웨어 업데이트 기능을 이용하여, 안드로이드 6.0 이상으로 업그레이드할 수 있는지 확인하고 업그레이드하시기 바랍니다.
안드로이드 OS를 업그레이드하는 경우 기존에 동의한 접근 권한 설정은 그대로 유지되므로 접근 권한을 다시 설정하려면 벅스 앱을 삭제하고 다시 설치해야 합니다.
